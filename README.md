# 📊 PhonePe Transaction Analysis (Power BI Project)

## 📌 Project Overview
This project focuses on analyzing PhonePe transaction data using Power BI to uncover insights about digital payment trends across India.

## 🎯 Objectives
- Analyze transaction patterns across states and districts
- Identify top-performing regions
- Understand growth trends in digital payments

## 🛠️ Tools Used
- Power BI
- Power Query
- Data Cleaning & Transformation

## 📊 Dashboard Features
- State-wise transaction analysis
- Year-wise growth trends
- Category-wise distribution
- Interactive filters for deep insights

## 💡 Key Insights
- Certain states dominate digital transactions
- Rapid growth observed in recent years
- Variation in category usage across regions
